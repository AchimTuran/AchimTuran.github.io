Achim's project portfolio

LXC (LibXConvolver)

LXC is a highly optimized cross platform open source library for fast convolution. The library has a simple API, which can be used for real application, prototyping or as a tutorial on how to use the included functions to develop more complex applications.

asplib (Achim's Signal Processing LIBrary)

This is my cross platform C/C++ signal processing library. At the moment it includes the following features:

  • A cross platform biquad filter class with factory design pattern to create filters with different processor optimizations
  • Frame buffer, frame ring buffer and normal buffer implementations through a C/C++ template class interface
  • An easy to use PortAudio interface class for rapid prototyping of C/C++ audio processing algorithms
  • The project is still under development and features where constanly added from time to time. For more informations just follow this link to my github.com repository.

adsp.template

It's a framework and starting point for developing new audio add-ons for Kodi's new Audio DSP extension point. I'm still improving the code during the development of adsp.biquad.filters and adsp.xconvolver.

adsp.xconvolver

adsp.xconvolver is convolution engine addon for Kodi's audio processing system. It's based on LibXConvolver, adsp.template and asplib.

adsp.biquad.filters

This project is also for Kodi's audio processing system. At the moment it only contains 10 band equalizer to adjust the frequency band levels.
